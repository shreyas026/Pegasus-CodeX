# Domestic Violence Case Analyzer (Frontend Prototype)

Next.js (App Router) + Tailwind CSS frontend with mock data only.

## Routes

- `/dashboard` - case overview and key metrics
- `/intake` - case intake form
- `/statement` - statement analysis input page
- `/result` - risk result summary and simple chart
- `/brief` - generated legal brief layout

## Run

```bash
npm install
npm run dev
```

## Notes

- This prototype is frontend-only.
- No backend, database, or API integration is implemented yet.
- All displayed values are mocked in `lib/mock-data.ts`.
